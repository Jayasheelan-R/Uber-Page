# Uber-Page

A small collection of static HTML pages created for learning and experimentation.

## Purpose

This repository is provided for educational and learning purposes only. It demonstrates simple static HTML layout and structure (pages, footers, images, and basic navigation) so you can study and modify the code.

Please avoid copying or publishing any copyrighted third-party material without permission. Replace any images, logos, text, or other assets with your own work or properly licensed resources before using this content in public or commercial projects.

## What's included

- Several example HTML pages (for example `main.html`, `map.htm`, `Ubereats.htm`, `cabs.htm`, etc.)
- A small `Images/` folder for page assets
- Footer and layout snippets (`footer.html`, `footermain.html`, `footerlist.html`, ...)

## How to view

1. Open the repository folder in your file browser or editor.
2. Open any of the `.html` files in a browser (for example, open `main.html`) by double-clicking the file or using your browser's "Open File" option.
3. Optional: if you prefer a local server for live reload or routing, use your editor's Live Server extension (for example, VS Code Live Server) or any simple static file server. No programming language is required to view these static HTML pages.

## How to use / modify

- Edit the `.html` files in a text editor or IDE.
- Replace images in `Images/` with your own or properly licensed images.
- Keep accessibility and semantic HTML practices in mind when making changes.
